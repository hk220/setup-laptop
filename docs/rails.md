# Rails Tips

## プロジェクトの作成

```bash
rails new [project name] -B
```
|オプション         |効果                                                 |
|:---------------:|:--------------------------------------------------:|
|ｰB, --skip-bundle|Railsプロジェクト作成時にbundle installを行わないようにする|
